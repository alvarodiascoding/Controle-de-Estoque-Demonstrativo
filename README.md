# Controle-de-Estoque-Demonstrativo
É um projeto simples feito a mão utilizando arquitetura de 3 camadas + DAO (intermediação com o banco) + Sistema (regras) + login e um banco de dados SQL Alchemy. Para a visualização utilizei PYSimpleGUI, mas recomendo que você use TKinter ou se possível refazer em outra linguagem.

O usuário e senha padrão são respectivamente AdmiN e 243546av.

Caso ainda tenha problemas, Comente o iniciador da função #login() e descomente o #executar().
